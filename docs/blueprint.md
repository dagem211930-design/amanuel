# **App Name**: PrimeFinish Pro

## Core Features:

- Project Cost Estimator: Allows users to input project area (SQ.M), select services (Ceramic, Painting, Gypsum), and choose between 'Full Supply' or 'Labor Only' options to generate an estimated project cost breakdown (Estimated Total, Labor 40%, Material 60%).
- Live Project Status Portal: Displays a visual, step-by-step progress tracker for active client projects, indicating completed, in-progress, and pending milestones through an interactive vertical timeline.
- Smart Project Progress Explanations: Leverages an AI tool to provide clear, concise explanations of each project milestone and offer proactive updates or answers to potential client FAQs directly within the status tracker, enhancing transparency and client understanding.
- Exclusive Projects Showcase: Presents a curated, responsive grid view of PrimeFinish Pro's past luxury projects, complete with high-resolution images, detailed descriptions, and showcased worker expertise like 'Marcus Aurelius' or 'Elena Petrov'.
- Direct Consultation Request: A streamlined form for prospective clients to schedule a consultation by providing their contact phone and preferred service via a dropdown, with a clear 'Book My Consultation' button to submit their request.
- Multilingual User Interface: Provides a user-friendly toggle to effortlessly switch the entire application's interface between Amharic and English languages, ensuring broad accessibility.

## Style Guidelines:

- Primary interactive color: A vibrant yet refined gold (#F5C01A) for call-to-action buttons, key highlights, and primary UI elements, embodying luxury and prestige against the dark background.
- Background color: A deep, sophisticated charcoal black (#121212) provides a premium dark theme base as specified, offering excellent contrast for golden accents and textual content.
- Accent color: A warm, rich bronze (#632C1F) is used for subtle distinctions, dividers, or complementary graphical elements, enhancing the luxurious palette with earthy undertones.
- Headlines: 'Alegreya' (serif) for an elegant, intellectual, and contemporary feel, enhancing the luxury branding and visual impact of titles.
- Body text: 'Inter' (sans-serif) for modern readability and a clean, objective appearance, ensuring clear communication against the dark background in various text blocks.
- Clean, minimalist, and vector-based icons are used throughout the application. Icons should have a metallic or gold-tinted appearance, reflecting modern luxury and sophistication.
- A highly responsive design using centered content blocks with consistent internal padding (e.g., 16px). Sections like the results card will prioritize perfect centering and avoid cutoff through adaptive layouts.
- Subtle and smooth transitions for state changes, section reveals, and hover effects are implemented. These animations will provide a refined and premium user experience without being distracting or excessive.
